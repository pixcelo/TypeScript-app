# Node.js Facebook login

## 環境
```bash
$ node -v
v20.11.0

$ npm -v
10.2.4
```

## npmインストール

インストール手順
```bash
$ npm init -y
```

```bash
$ npm install ejs
$ npm install express
$ npm install express-session
$ npm install passport
$ npm install passport-facebook
```

移動時
```bash
$ git pull
$ npm install
```
